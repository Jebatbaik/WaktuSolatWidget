Waktu Solat Widget (Windows)

Description:
A lightweight desktop widget for Windows that displays real-time Muslim prayer times for various zones in Malaysia. The data is fetched directly from the official JAKIM e-Solat portal.

Features:
Auto-updates prayer times from JAKIM every 4 hours.
Live digital clock and date in Bahasa Melayu.
Clean, transparent, dark-themed UI.
Remembers your selected zone upon restart.
Click and drag to move anywhere on your screen.

How to Install & Run:
Download the WaktuSolatWidget.exe file.
Double-click the file to open the widget. (No installation required!)
To make it run automatically when you turn on your PC:
Press Windows Key + R on your keyboard.
Type shell:startup and press Enter.
Right-click inside the folder that opens and select "Paste shortcut" (Make sure to create a shortcut of the .exe file; do not move the original file here).

How to Use:
Move: Click anywhere on the dark background and drag the widget to your preferred spot.
Change Location: Use the dropdown menu at the bottom to select your state/district. The widget will remember this choice the next time you open it.

Troubleshooting:
If the widget shows "Tiada sambungan internet", check your PC's internet connection. The widget will automatically try to reconnect.
Windows SmartScreen might flag the file as "Unrecognized" since it is a new custom app. Click "More info" and "Run anyway".
